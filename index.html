icantcode.today — Claude Code outage community

icantcode.today is a real-time developer community that activates exclusively when the Claude Code API is down. Claude Code API 장애 시에만 열리는 개발자 커뮤니티입니다.

How it works

The site polls the Claude Code API status every 30 seconds. When the API is up, only a landing page is shown. When the API goes down or degrades, a community feed opens immediately — no login required. Users get an anonymous UUID session. A nickname is only needed for posting, not for reading.

Key features

  • Real-time Claude Code API status monitoring (30-second polling)
  • Anonymous community feed — no account, no personal data collected
  • Terminal/CLI aesthetic design (MulmaruMono font, box-drawing characters)
  • Korean and English support
  • Dark and light mode

FAQ

Is Claude Code down right now?
Visit icantcode.today to see real-time status. When the API is down, the community feed opens automatically.
Is this official Anthropic service?
No. This is an independent community project by Claude Code users, not affiliated with Anthropic.
Do I need to sign up?
No. Anonymous UUID session is assigned automatically. Nickname only required to post.
icantcode.today가 뭔가요?
Claude Code API가 장애일 때만 열리는 개발자 커뮤니티입니다. 같은 처지의 개발자들과 소통하는 공간입니다.
로그인이 필요한가요?
아니요. 익명 세션이 자동 생성됩니다. 글을 쓸 때만 닉네임이 필요합니다.
